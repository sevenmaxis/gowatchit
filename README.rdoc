== GoWatchIt

A toy application to play with <tt>http://www.themoviedb.org</tt> and <tt>Google Play Movie</tt>

=== Notice

The application uses proxy server because of Google Play Movie restriction
in Ukraine region. So some links can be not fully functional.

== Getting Started

  git clone https://github.com/sevenmaxis/gowatchit.git
  cd gowatchit
  bundle
  rails s
  got to http://localhost:3000/
